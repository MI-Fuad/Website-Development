State 3 things that have inspired you when creating your website. These could include guest speakers, other websites, artists, developers etc.

When the large coursework was mentioned, I knew instantly what I wanted to make for my project, which is a portfolio of my own work as a photographer. I have been inspired by many photographers such as Daniel Kordan, Mario Testino, Hugo Lopes. Seeing their work and websites inspired me to make my own photo gallery. Parallax effect on web pages is something I found really interesting and appealing, so I used w3schools.com tutorial to help me achieve the outcome. Lightbox of images is something I found interesting too, after many attempts and adaptations with the w3schools.com tutotial, I was able to achieve my desired look.

State 3 ways in which your site is accessible.

I have used hedings correctly to organise the structure of my content

'alt="" ' tags on images with desriptions where appropriate; in case of images not loading, user would be able to see brief description on the image

I have made buttons and links with unique and appropriate names which makes users navigate the website with ease making the experience more user-friendly for different categories of users. Mostly natively usable from keyboard

State 3 ways in which you have considered usability in your site.

It is a website related to photography, that message is given to users when site loads. I have tried to layout in a way that the users are able to guess what the page is about almost immediately.

I have included arrows where appropriate to navigate my site, for example the arrown on homepage for users to understand scroll down for more content. It gives a nice flow to the website in my opinion.

Consistent look throughout the website using same colour schemes for the same type of content.

State 3 things you had to learn/find out by yourself when creating your site, and describe how you did that (i.e. what searches you did, any new tools/techniques you learned, any changes/adaptations you made to a particular resource to make it work with your site).

Parallax Effect - learnt how to give the site a parallax scrolling effect, using nothing but pure CSS style codes.

Masorny Layout - Learnt about how to layout images in a masorny layout without any javascript but completely with CSS style codes.

Modal Lightbox - Learnt about how to use a combination of javascript and CSS to create a Lightbox image gallery.

What aspect(s) of your work do you think worked well and why?

I believe the parallax effect of my work and the overall design worked well because I kept it quite minimal and simple with suitable colour schemes accross the site, giving it overall a consistant look and feel.

Responsiveness of my site is another aspect works well as I have used media queries for different sized screens

What aspect(s) of your work could be improved, and how might you do things differently another time?

Accessability of my website could be improved, by making the entire website usable through the keyboard and make it more usable for disable users.

Html code for images could be improved using templates.
